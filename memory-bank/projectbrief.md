# Project Brief

## Overview
A minimal, modern portfolio website to showcase projects created at Gauntlet AI, featuring video demonstrations and resource links. The site is built with Astro and Tailwind CSS, designed to be fast, responsive, and elegant.

## Core Goals
- **Showcase Projects**: Display web, mobile, and desktop applications effectively using video thumbnails and modal details.
- **User Experience**: Provide a simple, elegant, and fast browsing experience.
- **Performance**: Achieve high Lighthouse scores (>90) with static site generation.
- **Design**: Implement a clean, minimal aesthetic with light/dark mode support.

## Key Features
- **Homepage**: Responsive grid of project cards.
- **Project Showcase**: Interactive cards with video previews and detailed modals.
- **Navigation**: Fixed header with About/Contact links and theme toggle.
- **Pages**: About (professional info), Contact (form).
- **Tech Stack**: Astro, Tailwind CSS, TypeScript, React/Preact (interactive parts).

## Success Criteria
- Fully responsive design across devices.
- Fast load times and smooth transitions.
- Functional contact form and navigation.
- Accessibility compliance (WCAG 2.1 AA).
