# Project Brief: Andrew Shindyapin Portfolio

## Project Overview
A minimal, modern portfolio website showcasing projects created at Gauntlet AI, featuring clean design, video demonstrations, and comprehensive project information.

## Core Requirements
- **Purpose**: Professional portfolio to showcase web/mobile/desktop applications
- **Target Audience**: Potential employers, colleagues, collaborators, professional network
- **Hosting**: GitHub Pages (static site)
- **Performance**: Lighthouse score > 90 for all metrics

## Key Features
1. **Project Showcase**: Grid of project cards with video thumbnails and modals
2. **About Section**: Professional bio with social links
3. **Contact Form**: Simple form with validation
4. **Theme Support**: Light/dark mode with system preference detection
5. **Responsive Design**: Mobile-first, works across all devices

## Technical Constraints
- Static site generation (no SSR)
- Fast load times with lazy-loaded content
- SEO-friendly with proper meta tags
- WCAG 2.1 AA accessibility compliance

## Success Criteria
- Clean, minimal aesthetic
- Fast performance (sub-3s load time)
- Professional presentation of work
- Easy for visitors to contact and learn more
